工具：HTML 表格提取器 这是我不断增长的粘贴转换工具集合中的又一新成员。该工具接受来自浏览器的粘贴富文本（包含嵌入式 HTML 表格），并将检测到的每个表格转换为 HTML、Markdown、CSV、TSV 或 JSON 格式。您可以尝试选择维基百科上"旧金山湾区城市和城镇列表"页面的所有内容，然后直接粘贴到该工具中：类似地，我最近重建了"富文本转 Markdown"工具，增加了对表格的支持并全面改进了用户界面。更新：事实证明，维基百科有一个开放的 CORS API，可用于检索任何页面的完整渲染 HTML 内容——演示在此——因此我让 Codex 添加了搜索维基百科页面并自动导入和显示该页面中所有表格的功能。标签：html, tools, wikipedia, cors
